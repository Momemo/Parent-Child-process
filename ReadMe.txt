- To compile pre.c: gcc -o pre pre.c 
- To Run pre.c: ./pre

- To compile sort.c: gcc -o sort sort.c
- To run sort.c: ./sort

- To run together ./pre | ./sort


- To compile cpipe.c: gcc -o pipe cpipe.c
- To run: ./pipe

- To compile unixcmnd: gcc -o prog3 unixcmnd.c
- To run unixcmnd:  "./prog3 ls" ***ls is just the example it can be any command**

** NOTE **

When running these files there will be no prompt to enter information. Once you run please enter the Name and GPA and CONTRL+D to end it. I have provided screenshots for reference.
